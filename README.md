# BlackJack

My own version of the popular game [Blackjack](https://en.wikipedia.org/wiki/Blackjack)<br>
This version will have a few tweaks like;

- [ ] Make the already played cards append to the end of the deck e.g 1,2...51,52 -> 4,5,6...52,1,2 
- [ ] Change the looks of the deck, have light and dark versions.
- [ ] Change the break value from 21 to x
- [ ] Online mode to play with friends (far fetched)


For this I'll be using "pygame", a python library made for game development.
